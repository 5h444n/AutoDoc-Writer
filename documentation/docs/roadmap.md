# Project Roadmap

**Methodology:** Extreme Programming (XP) | **Duration:** Nov 2025 - Feb 2026

## Timeline

### Sprint 1: Foundation & Connectivity
* **Focus:** UI Skeleton, Basic GitHub Auth, Repository Fetching.
* **Status:** Setup Electron + React environment and FastAPI server.

### Sprint 2: The Intelligence Core
* **Focus:** AI Integration (Basic Summary) and Active Monitoring toggle.
* **Key Feature:** "Detect New Commits" polling logic.

### Sprint 3: Heavy Lifting (Crunch Sprint)
* **Focus:** Deep Logic (LaTeX/Research generation) and Export options.
* **Key Tasks:**
    * Test LaTeX in Overleaf.
    * Implement "Diff Reading" to analyze only changed files.

### Sprint 4: Polish & Offline Capability
* **Focus:** Offline Mode (Caching), Settings, and Packaging.
* **Key Feature:** Load data from local cache if API fails.

### Sprint 5: Final Demo Prep
* **Focus:** Full Product Demo and Bug Bash.
* **Final Step:** Create installer (`.exe` / `.dmg`) using electron-builder.
